# AI Rosary Habit Companion
**An 80-Day Spiritual Transformation Agent**

## 🚀 The Mission
Consistency is the hardest part of building a spiritual habit. This project moves beyond static reminders by architecting an automated system that integrates daily prayer into a digital workflow. It is designed to support an 80-day Rosary challenge through proactive scheduling and data ownership.

## 🛠️ The Tech Stack
- **Python & Streamlit**: A mobile-friendly dashboard for daily "check-ins."
- **Google Calendar API**: Proactive integration that syncs the 80-day plan with my personal schedule.
- **SQLite (The Habit Ledger)**: A local database that tracks every "Completed" day, ensuring 100% data privacy and ownership.

## 📈 Key Features
- **Smart Scheduling**: Generates an 80-day calendar series with 20-minute time blocks.
- **Progress Tracking**: A live progress bar that visualizes the journey from Day 1 to Day 80.
- **The "Success" Loop**: An interactive "Mark Completed" button that triggers celebratory UI elements (balloons) and updates the permanent ledger.

## 🏁 Why This Matters
As a Data Scientist, I often build models for corporate optimization (like my work with **MTN Churn Prediction**). This project represents a shift toward **Personal AI Governance**—using the same high-level tools to optimize my own discipline and spiritual growth.
